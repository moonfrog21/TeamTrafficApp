# TeamTrafficApp 🚦

## Overview & Purpose
TeamTrafficApp is a cross-platform mobile prototype developed for both iOS and Android using dart in Flutter. This application was designed specifically for the IEEE IAS Design Competition to function as mobile app.

The overall Traffic management system aims to control traffic lights via IoT devices, live, based on data from user's map app data.

## current prototype
sends user GPS data via a secure tunnel to locally hosted server.
The user specifies the server(saved in memory) and clicks a button to begin sending live GPS data
